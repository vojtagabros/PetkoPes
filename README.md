# Petko - Dog Walk Tracker

A simple, mobile-friendly web app for tracking your dog walks.

**Live demo:** https://vojtagabros.github.io/PetkoPes/

## Features

- **Walk Timer** – Start, pause, resume, and finish walks with a real-time timer
- **Notes** – Add notes to each walk (e.g. route, weather, dog's mood)
- **Walk History** – Browse all past walks with duration and timestamps
- **Stats** – See total walks, total time, and walks this week
- **Offline Storage** – All data is saved locally in your browser (localStorage)

## Usage

No build step or server required. Just open `index.html` in your browser.

To deploy on GitHub Pages:

1. Push this repo to GitHub
2. Go to **Settings > Pages**
3. Set source to **main branch** and root folder
4. Your app will be live at `https://<username>.github.io/<repo-name>/`

## Tech

Single HTML file with inline CSS and vanilla JavaScript. No dependencies.

## Screenshots

Open `index.html` on your phone or use your browser's mobile view (DevTools > Toggle Device Toolbar) for the best experience.
